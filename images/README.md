# Screenshots da Landing Page

Adicione as seguintes imagens nesta pasta:

## Imagens Necessarias

1. **screenshot-correcao.png** - Screenshot da tela de correcao de redacao
   - Mostrar o formulario de entrada e o resultado da correcao
   - Destacar as cores de erros (declinacao, conjugacao, etc.)

2. **screenshot-progresso.png** - Screenshot do dashboard de progresso
   - Mostrar o grafico de pizza de distribuicao de erros
   - Mostrar o historico de erros em barras

3. **screenshot-vocabulario.png** - Screenshot das listas de vocabulario
   - Mostrar as listas criadas (ex: Antigas, Reuniao VW)
   - Mostrar palavras com traducao e nivel de dificuldade

4. **screenshot-flashcards.png** - Screenshot do jogo de flashcards
   - Mostrar a configuracao de filtro por dificuldade
   - Vermelho (dificeis), Amarelo (medios), Verde (faceis)

## Formato Recomendado

- Formato: PNG
- Resolucao minima: 1200px de largura
- Fundo escuro (tema dark do sistema)
